Using the NER tagger

1.) Have NLTK installed. Download the 'punkt' extension (this is commented out)

2.) Set the java_path variable to your machines jdk directory (the one in there right now is mine as an example)

3.) Set the StanfordNerTagger functions parameters to your directory of the english.all.3class.distsim.crf.ser.gz file followed by the directory of the stanford-ner.jar

4.) Run it
